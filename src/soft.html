Программное обеспечение SelfyCheck Diabetes Management System — это аналитический сервис, который позволяет скачивать показания глюкометра, сохранять их в системе SelfyCheck Diabetes Management System и затем анализировать с помощью графических и других инструментов. Благодаря информации, получаемой в результате анализа данных в системе SelfyCheck Diabetes Management System, можно лучше контролировать уровень глюкозы в крови.

Программное обеспечение выполняет следующие функции:

  • передача данных об уровне глюкозы в крови из прибора на ПК
  • использование цифровых данных вместо записей в журнале
  • отображение тенденций и статистики по результатам измерений
  • предоставление специалистам результатов измерений.

Программное обеспечение для анализа данных помогает:

  • пыстро получить данные об уровне глюкозы крови
  • получить более глубокое понимание тенденций изменения глюкозы
  • в редиме динамики наблюдать за изменениями уровня гликемии
  • формировать базу показателей гликемического контроля
  • выявить временные промежутки, в которые гликемия отличается от целевых значений
  • акцентировать внимание на состояние гликемического профиля и мотивации на достижение целей гликемического контроля
  • распечатать результаты самоконтроля для информирования лечащего врача
  • дистанционно оценить уровня гликемии

Алгоритм инсталляции программного обеспечения

Чтобы установить ПО SelfyCheck Diabetes Management System:

1. Включите компьютер.

2. Закройте все фоновые программы, особенно экранные заставки и антивирусы, так как они могут вызвать проблемы во время установки.

3. Распакуйте скачанный файл (с помощью WinRAR, 7zip). На экране ПК откроется каталог с программой SelfyCheck Diabetes Management System.

4. Запустите установку программы с помощью файла «SELFYcheck Diabetes Management System». Нажмите кнопку Next/Далее.

5. Откроется окно мастера установки ПО SelfyCheck Diabetes Management System. Нажмите Next/Далее для продолжения установки. Если нажать кнопку Cancel (Отмена), процесс установки будет остановлен.

6. Укажите каталог для установки программы в окне выбора каталога. Выберите предложенный для установки каталог или нажмите Browse/Обзор для выбора другого. Выберите вариант Everyone/Для всех. Затем нажмите кнопку Next/Далее.

7. Появится окно Confirm Installation /Подтверждение установки. Нажмите кнопку Next/Далее для запуска процесса установки. Для установки программы необходимы права администратора.

8. Если установка будет выполнена успешно, появится окно Installation Complete/Установка завершена. Нажмите кнопку Close/Закрыть для выхода из программы установки.

9. На рабочем столе появится ярлык SelfyCheck Diabetes Management System, а программа SelfyCheck Diabetes Management System будет добавлена в список программ ПК
